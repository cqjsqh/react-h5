#####开发

npm install
npm run dev

#####目录
```
api -----------------------数据接口
assets -----------------------资源
components ----------------------组件
constants ----------------------------常量
containers --------------容器,调用组件组织页面(组件和路由的组合体)
actions -----------------------用户交互,view通知state变化
reducers -----------state的状态机,收到action以后，给出一个新的state
store -------------------------数据仓库
main.js ---------------------------项目入口
```
